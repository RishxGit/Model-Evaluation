# Model Evaluation

This repository is a personal sandbox dedicated to practicing model evaluation techniques, tuning hyperparameters, and avoiding overfitting.

## 🧠 What's Inside
* **`model_evaluation.ipynb`**: Core notebook containing the evaluation workflows and metrics.
* **Automated Tuning**: An automated loop implemented to test different hyperparameter combinations and select the optimal model configuration.
* **Regularization Optimization**: Experiments focused on finding the ideal regularization strength to balance bias and variance.

## 🛠️ Concepts
Cross-validation, Hyperparameter Tuning (Grid/Random Search), Overfitting vs. Underfitting, and Evaluation Metrics.
